# 匯率試算工具 - 設計概念

## 選定設計方案：現代金融工具風格

### 設計哲學
**Modern Financial Dashboard** - 簡潔、精準、專業的金融工具介面風格。強調數據的清晰展示與快速比較，採用冷靜的色調與充足的留白，讓用戶能夠專注於核心計算結果。

### 核心原則
1. **數據優先** - 所有設計決策圍繞數據清晰度展開，避免視覺干擾
2. **快速掃描** - 採用對比鮮明的色彩標記，讓用戶快速識別最優方案
3. **信任感** - 使用專業的排版與精確的間距，傳達可靠性
4. **漸進式揭示** - 基礎計算在上方，詳細比較在下方，避免認知負荷

### 色彩哲學
- **主色** - 深藍 (#1e40af) 代表金融穩定性與信任
- **對比色** - 翠綠 (#059669) 用於標記最優結果（最高收益）
- **警告色** - 琥珀 (#d97706) 用於標記較差結果
- **背景** - 清爽白色 (#ffffff) 與淺灰 (#f9fafb)
- **文字** - 深灰 (#1f2937) 確保高可讀性

### 排版系統
- **標題** - Poppins Bold (700) 用於主標題，傳達現代感
- **副標題** - Inter SemiBold (600) 用於分類標題
- **正文** - Inter Regular (400) 用於數據與說明
- **數據** - Inter Mono (等寬字體) 用於金額與匯率，確保對齊

### 版面設計
- **上部分** - 輸入表單區域，簡潔的卡片設計
- **中部分** - 計算結果摘要，大號數字突出顯示
- **下部分** - 銀行比較表，採用行交替背景色提升可讀性
- **側邊** - 無側邊欄，全寬設計最大化表格空間

### 簽名元素
1. **漸變分割線** - 在各區域間使用微妙的漸變分割線
2. **數據卡片** - 採用陰影與邊框的卡片設計，突出重要數據
3. **對比標記** - 使用背景色與圖標標記最優與最差結果

### 互動設計
- **輸入框** - 聚焦時顯示藍色邊框與微妙陰影
- **懸停效果** - 表格行懸停時顯示淺灰背景
- **複製功能** - 點擊數據可複製到剪貼板，顯示簡短提示
- **響應式** - 移動端表格採用卡片堆疊設計

### 動畫指南
- **進入動畫** - 計算結果淡入 (200ms)
- **過渡效果** - 所有顏色變化使用 150ms 過渡
- **加載狀態** - 計算中顯示微妙的脈衝動畫

### 無障礙設計
- 所有文字對比度 ≥ 4.5:1
- 表單標籤明確關聯
- 鍵盤導航完全支持
- 屏幕閱讀器友好的 ARIA 標籤
